tools and technologies

We will use IntelliJ IDEA to build our project. In IDEA, we will use the maven repository to add our project dependencies in order to ensure project portability.
This application requires that it can be used on the mobile phone, so we need to develop the application based on the API plug-in of the Android mobile phone.
We will use java and python as our development languages. At the same time we will use Photoshop to make our UI interface design.
